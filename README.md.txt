# Naitik Shakya Portfolio

This is the personal portfolio website of **Naitik Shakya**, an aspiring game designer.

## Features
- Dark theme with Electric Cyan accents
- Clean typography with Montserrat & Inter
- Smooth scroll animations with AOS
- Responsive grid for projects
- Scroll-to-top button

## How to Use
1. Clone or download this repo.
2. Open `index.html` in your browser.
3. Deploy using GitHub Pages or any static hosting.
